# Graficadora de Funciones Lineales

Una página web interactiva para visualizar funciones lineales de la forma **y = m·x + b**.

Ajusta la pendiente (m) y la intersección con el eje Y (b) para ver cómo cambia la recta en tiempo real.

## Características

- Controles para pendiente (m), intersección (b) y color de la recta
- Gráfico interactivo con cuadrícula y ejes
- Información en tiempo real de la ecuación y coordenadas
- Diseño responsivo

## Tecnologías

- HTML5 + CSS3 + JavaScript (Canvas API)
